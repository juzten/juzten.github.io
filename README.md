# Juzten.com
### A Portfolio Website

[www.juzten.com](https://www.juzten.com)

## Project setup and commands
```
npm install
```
Run the dev server
```
gulp dev
```
Run all build commands
```
gulp build
```
Build images
```
gulp images
```
Build styles
```
gulp styles
```
Build scripts
```
gulp scripts
```
